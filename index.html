Featured Work: Lenovo AI Optimization

Transforming a 10GB raw model into a 1GB high-speed production tool.

PaddleOCR ONNX Runtime FastAPI

The Challenge

Lenovo needed to process 5,000+ gaming images for app state detection. The original process relied on heavy models and manual verification, which was slow and resource-intensive.

The Solution

I engineered an automated testing pipeline using PaddleOCR and YOLOv11. I fine-tuned the model for app-launch detection and exported it using ONNX Runtime to maximize inference speed. Finally, I packaged the entire Python environment into a standalone Windows executable.
